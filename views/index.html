In this blog I am going to explain about Ripple XRP Block Chain. This blog is for developer who is interested to learn Ripple XRP Blockchian.

Following the operation you can perform on Ripple XRP API
.
  1. Wallet Address Generation.
  2. Get Balance.
  3. Send or Transfer XRP
  4. Check Transaction Status
  5. Multiple Siging.

1. Wallet & Address Generation

XRP Wallet Generation is Offline process no need to communicate with any XRP Server. We can generate address offline and load some test xrp from other address. The full code available in the [github] You can download. Please make a note of Clasic Address, Seed, Secrete Key, public and private Key for next set of API.

2. Check Balance.

Using this API you can check the balance of any testnet xrp address.

3. Fund/Transfer Some XRP

The XRP Address you have generated from Secrete Key is offline process. You need to fund some XRP from existing XRP account. Otherwise You need to generate new xrp address from Ripple Testnet XRP and generate TestNet Credentials. This will give you an address with Test XRP as balance. You can use this TestNet XRP as from account to fund your own generated XRPs from seed.
